# BNO085 Orientation Cube

Displays BNO085 orientation as a 3D cube. Quaternion data is used when available, with Euler angles as a fallback.

## Requirements

- Configure the board OSC destination to the computer running Processing.
- Enable BNO085 quaternion output, or enable BNO085 Euler output as a fallback.
- Install the Processing oscP5 library.
