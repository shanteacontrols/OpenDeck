# VL53L5CX Heatmap

Draws the VL53L5CX 8x8 distance grid as a color heatmap. Close objects are highlighted so hand movement is easy to see.

## Requirements

- Configure the board OSC destination to the computer running Processing.
- Set VL53L5CX output mode to **Grid**.
- Set VL53L5CX resolution to 8x8.
- Install the Processing oscP5 library.
