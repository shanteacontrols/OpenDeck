# VL53L5CX Touchless Synth

Turns the VL53L5CX 8x8 grid into a simple touchless instrument. Hand position and distance drive the visual pads and generated sound.

## Requirements

- Configure the board OSC destination to the computer running Processing.
- Set VL53L5CX output mode to **Grid**.
- Set VL53L5CX resolution to 8x8.
- Install the Processing oscP5 library.
- Install the Processing Sound library.
