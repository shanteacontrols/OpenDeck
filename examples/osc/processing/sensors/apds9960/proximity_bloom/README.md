# APDS9960 Proximity Bloom

Visualizes APDS9960 proximity as a glowing bloom. Move a hand or object near the sensor to expand the flower, brighten the core, and energize the particles.

## Requirements

- Configure the board OSC destination to the computer running Processing.
- APDS9960 **Proximity/gesture mode** set to **Proximity**.
- Install the Processing oscP5 library.
