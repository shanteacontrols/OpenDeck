# APDS9960 Ambient Light Theater

Turns APDS9960 ambient light readings into a window-lit room. Covering or exposing the sensor dims or brightens the scene, opens the blinds visually, and changes the dust motion.

## Requirements

- Configure the board OSC destination to the computer running Processing.
- APDS9960 **Ambient light** enabled.
- Install the Processing oscP5 library.
