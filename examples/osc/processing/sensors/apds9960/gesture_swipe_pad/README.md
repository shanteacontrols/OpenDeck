# APDS9960 Gesture Swipe Pad

Visualizes APDS9960 gesture events as a stack of swipeable cards. Each detected gesture throws the top card up, down, left, or right and advances the deck like a phone or tablet interaction.

## Requirements

- Configure the board OSC destination to the computer running Processing.
- APDS9960 **Proximity/gesture mode** set to **Gesture**.
- Install the Processing oscP5 library.
