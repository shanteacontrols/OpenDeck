# VL53L4CX Touchless Resonator

Turns the VL53L4CX normalized distance output into a soft touchless resonator. Slow hand movement gently plays the current note area, while sharper movement adds accents; press space to mute.

## Requirements

- Configure the board OSC destination to the computer running Processing.
- Enable VL53L4CX normalized distance output.
- Install the Processing oscP5 library.
- Install the Processing Sound library.
