# VL53L4CX Distance Tunnel

Visualizes the VL53L4CX single-zone distance reading as a responsive depth tunnel. Press `m` to drive the effect from raw millimeters, or `n` to use the firmware-normalized output.

## Requirements

- Configure the board OSC destination to the computer running Processing.
- Enable VL53L4CX normalized distance output.
- Enable VL53L4CX distance mm output only if you want to use raw millimeter mode.
- Install the Processing oscP5 library.
