# CAP1188 Light Console

Turns the eight CAP1188 capacitive touch inputs into a small lighting desk. Each touch toggles one colored beam, so the sensor behaves like eight physical scene buttons.

## Requirements

- Configure the board OSC destination to the computer running Processing.
- CAP1188 touch OSC output on `/opendeck/sensors/cap1188/touch/<0..7>`.
- Install the Processing oscP5 library.
