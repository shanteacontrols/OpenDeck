1 # Core modules
2 
3 Collection of modules for easier control of integrated AVR and STM32 peripherals, as well as some general helping macros and symbols.
4 
5 This collection is split into two parts:
6 
7 - General
8 - HAL
9 
10 ## General
11 
12 This directory contains several modules which are independent of underlying architecture:
13 
14 - Bit manipulation
15  - Macros for easier bit access in variables and registers.
16 - Ring buffer
17  - Contains lightweight ring buffer implementation from LUFA project.
18 - Strings
19  - C++ class used for easier building of strings. `USE_STRINGS` symbol must be defined for compilation and `STRING_BUFFER_SIZE` symbol must be defined with wanted value, for instance 50: `STRING_BUFFER_SIZE=50`
20 - Timing
21  - Includes simple delay function for AVR architecture, as well as `rTimeMs()` function used to get current run time in milliseconds. This function uses `rTime_ms` variable which must be externally implemented and updated in order to use the function.
22 - Misc
23  - Includes various helper constants and macros.
24 
25 ## HAL
26 
27 This directory contains some hardware abstraction APIs and macros for AVR architecture, as well as STM32 ARM microcontrollers which are using CubeMX HAL. For these microcontrollers, this module is just a wrapper around CubeMX HAL.
28 
29 ### AVR
30 
31 Contains following modules:
32 
33 - ADC
34  - Used for easier access to ADC peripheral inside AVR.
35 - Pin manipulation
36  - Used for easier pin access (read/write and setting input/output).
37 - Reset
38  - Module used to initiate MCU reboot.
39 - SPI
40  - Used for easier access to SPI peripheral inside AVR.
41 - Interrupt
42  - Simple macros used to disable and enable all interrupts.
43 
44 Only ADC module requires configuration.
45 
46 #### ADC
47 
48 To configure ADC, structure of `adcConf` type must be intitialized. Structure holds two elements: `prescaler` and `vref`.
49 
50 ##### Prescaler
51 
52 Valid options are 128, 64, 32 and 16. To define prescaler, use one of the following options for `prescaler` element:
53 - `ADC_PRESCALER=128`
54 - `ADC_PRESCALER=64`
55 - `ADC_PRESCALER=32`
56 - `ADC_PRESCALER=16`
57 
58 ##### Analog reference
59 
60 Reference can be set to internal 1.1V, internal 2.56V, AVCC reference or AREF. To define reference, set `vref` element to one of the following values:
61 
62 - `VREF_AREF`
63 - `VREF_AVCC`
64 - `VREF_INTERNAL_2V56`
65 - `VREF_INTERNAL_1V1`
66 
67 ##### Enabling ADC
68 
69 In order to actually enable ADC, additional symbol must be defined.
70 
71 - `USE_ADC`
72 
73 ### STM32
74 
75 In order to use this module, CubeMX HAL must be used and `STM32` symbol must be defined.
76 
77 - Pin manipulation
78  - Used for easier pin access (read/write only).
79 - Interrupt
80  - Simple macros used to disable and enable all interrupts.
